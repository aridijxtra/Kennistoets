# Langdurige Gedragsverandering – Coach Quiz

Een interactieve quiz voor leefstijlcoaches over langdurige gedragsverandering.

## Over deze website

Deze website test kennis over langdurige leefstijlverandering aan de hand van **10 realistische coachinscases**. De gebruiker maakt per case een keuze en krijgt aan het einde persoonlijke feedback op de totaalscore.

## Kenmerken

- ✅ 10 interactieve cases uit de coachingspraktijk
- ✅ Direct feedback na elke keuze (goed/fout)
- ✅ Persoonlijke eindscore met feedback
- ✅ Verwijzing naar *Het Praktijkboek voor de Leefstijlcoach*
- ✅ **Geen database, geen opslag van gegevens** – alles blijft op het scherm van de gebruiker
- ✅ Volledig responsief (mobiel & desktop)
- ✅ Eén enkel HTML-bestand, geen externe afhankelijkheden (behalve Google Fonts)

## Publiceren op GitHub Pages

1. Maak een nieuw repository aan op [github.com](https://github.com)
2. Upload `index.html` naar het repository
3. Ga naar **Settings → Pages**
4. Kies bij *Source*: `Deploy from a branch` → `main` → `/ (root)`
5. Klik op **Save**
6. Na een minuutje is de site live op `https://jouwgebruikersnaam.github.io/repositorynaam/`

## Techniek

- Puur HTML, CSS en JavaScript – geen frameworks, geen build-stap
- Lettertypen via Google Fonts (DM Serif Display + DM Sans)
- Werkt direct vanuit de browser, ook offline na eerste keer laden

## Privacy

Deze website slaat **geen gegevens op**. Er is geen database, geen tracking, geen cookies. Alle interactie vindt uitsluitend in de browser van de gebruiker plaats.
